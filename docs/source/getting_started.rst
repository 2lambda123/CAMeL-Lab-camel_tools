Getting Started
===============

Installation
------------

You will need Python 2.7 or Python 3.4 and above.

Using pip
^^^^^^^^^

.. code-block:: bash

   pip install camel_tools

From Source
^^^^^^^^^^^

.. code-block:: bash

   # Download the repo
   git clone https://github.com/owo/camel_tools.git
   cd camel_tools

   # Install CAMeL Tools and all dependencies
   pip install .
